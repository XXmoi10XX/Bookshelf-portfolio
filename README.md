# 概要
改訂4版 基礎 Ruby on Railsで作ったアプリに手を加えたものです。
本の登録、レビュー、検索が出来ます。

# 制作背景
改訂4版 基礎 Ruby on Railsで作ったアプリに手を加えたものです。 
本で学習した知識を実際に自分で考えた機能を付け加える事で理解を深めると同時にポートフォリオとして
利用するために制作しました。

# 機能一覧 

### 本一覧、会員の一覧ページ
- 検索機能
- 詳細ページ
- 新規登録・更新
- バリデーション

### 認証機能
- sessionを使ったログイン
- パスワード変更
 
### 投稿機能
